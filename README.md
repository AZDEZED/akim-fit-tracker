# Akim Fit Tracker V2

Application web mobile-first pour suivre poids, nutrition et entraînements.

## Installation GitHub Pages

1. Créer un nouveau dépôt GitHub nommé `akim-fit-tracker`.
2. Ajouter ces fichiers à la racine du dépôt :
   - index.html
   - manifest.json
   - sw.js
   - icon.svg
3. Aller dans Settings > Pages.
4. Dans Build and deployment, choisir :
   - Source : Deploy from a branch
   - Branch : main
   - Folder : /root
5. Ouvrir l'URL GitHub Pages générée depuis Chrome Android.
6. Menu ⋮ > Ajouter à l'écran d'accueil.

Les données sont stockées localement dans le téléphone. Utiliser Export JSON régulièrement.
